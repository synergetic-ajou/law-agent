# ⚖️ 내 손안의 AI 변호사
**AI 챗봇 기반 소액 소비자 피해 구제 솔루션**  
개인 간 전자상거래(C2C)에서 발생한 소액 피해를 AI로 분석 → 절차 추천 → 문서 자동 작성 → 기관 신청 안내까지 지원하는 법률 조력 서비스입니다.

---

##  프로젝트 소개
중고거래·SNS 계좌거래 등 개인 간 전자상거래(C2C)는 급증하고 있지만,  
현행 전자상거래법은 사업자–소비자 관계를 전제로 하고 있어 실제 피해 대응이 어렵습니다.

**예시:** 문고리 거래, 인스타 DM 거래, 계좌송금 직거래 등  

이 프로젝트는 이러한 **법적 사각지대 피해자**가 겪는 절차적 어려움을 해결하기 위해 설계되었습니다.

---

##  목표
- 소액 피해자의 **법률 접근성 향상**  
- **법률 문서 자동화** (내용증명, 조정 신청서, 고소장 등)  
- 피해 상황에 맞는 **절차적 의사결정 안내**  
- 대한민국 현행 법체계 내에서 가능한 **합법적 구제 프로세스 자동화**

---

##  시스템 아키텍처

### 1) 문서 임베딩·DB 구축 파이프라인
- PDF 법령 문서를 파싱 → 조 단위 청킹 → PGvector에 저장 → RAG 기반 챗봇 검색 가능
- **기술:**  
  - PyMuPDF 기반 PDF 텍스트 추출  
  - 정규식 기반 구조 분석 (편/장/절/조)  
  - SentenceTransformer 임베딩 생성  
  - PostgreSQL + PGvector 저장

### 2) AI 챗봇 백엔드
- FastAPI 기반  
- RAG(Retrieval-Augmented Generation) 활용  
- 사용자의 피해 상황을 구조화하여 **적절한 절차 추천**

### 3) 법률 조력 프로세스 엔진
- **내용증명 자동 작성**  
- **사기 의심 판단 및 형사·민사 절차 안내**

---

##  서비스 구제 프로세스
본 서비스는 대한민국 법 체계에 기반하여 다음 절차를 자동화합니다.

1️⃣ **내용증명 작성·발송 지원** (비공식 조정 단계)  
- 계약 불이행, 환불 거부 등 초기 분쟁 상황 정리  
- AI가 내용증명 자동 작성  

2️⃣ **전자거래분쟁조정위원회(ECMC) 신청 안내**  
- 조정 신청에 필요한 내용 자동 구성  
- 신청서 초안 자동 생성  
- 제출 절차 안내  

3️⃣ **형사 절차 지원** (사기 의심 시)  
- 사기죄 성립 요건 체크  
- 고소장 자동 작성  
- 증거 정리 방법 안내  

4️⃣ **민사 절차 지원** (소액심판청구)  
- 미회복 피해금 대상 소액민사청구 지원  
- 소송 서류 기본 골격 자동화

---

##  처리 파이프라인 요약
1. 법령 PDF 수집  
2. 텍스트 파싱  
3. 조 단위 청킹  
4. SentenceTransformer로 임베딩 생성  
5. PGvector에 저장  
6. 챗봇에서 사용자의 질의와 임베딩 매칭 → 정확한 법률 조언 제공

---

##  프로젝트의 의의
- C2C 전자상거래 피해의 **법적 사각지대 해소**  
- 변호사 접근이 어려운 **소액 피해자에게 실질적 법률 지원 제공**  
- 법령 기반 RAG 모델로 **부정확한 AI 응답 방지**  
- 단순 안내에 그치지 않고 **문서 자동 작성까지 지원**
